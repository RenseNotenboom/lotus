# Lotus

Lotus is a mobile-first sleep tracker for newborns and babies (0-12 months).

## MVP Features

- One-tap sleep timer (start/stop)
- Manual sleep entry with validation and overlap guard
- Age-based nap recommendation with bounded personalization
- Dashboard with quick actions
- Timeline and history summary helpers
- Local reminder scheduling service

## Tech

- Expo React Native (TypeScript)
- Jest + React Native Testing Library

## Scripts

- `npm install`
- `npm test -- --runInBand`
- `npm run start`
- `npm run web`

## Current Status

Core MVP foundations are in place and covered by tests.
